# ТЗ-1

В рамках этого технического задания был создан файл Excel с опросником, включающим вопросы с ответами "да" или "нет". Данные были сгруппированы по месяцам и вопросам, а также подсчитано количество ответов "да" и "нет". На основе этих данных был построен график, демонстрирующий изменение количества ответов "да" на два вопроса:

1. Был ли сделан заказ с Директором Партнером?
2. Была ли сделана корректировка страховых запасов через систему Pandas?

Из графика видно, что в июне количество ответов "да" на вопрос о заказе с Директором Партнером уменьшилось, в то время как количество ответов "да" на вопрос о корректировке страховых запасов через систему Pandas увеличилось.

Обратите внимание, что все данные являются вымышленными и были использованы исключительно для выполнения данного задания.
